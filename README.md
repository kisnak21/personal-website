# Kresna Portfolio

A modern, responsive portfolio website built with React and Vite to showcase projects, skills, and experience.

## Features

- **Home** - Introduction and overview
- **Projects** - Showcase of development projects with details
- **Skills** - Technical skills with proficiency visualization
- **Responsive Design** - Works on desktop and mobile devices

## Tech Stack

- React 18
- Vite
- ESLint
- Modern CSS (CSS Variables, Flexbox, Grid)

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── data/           # Static data (projects, skills)
├── pages/          # Page components
└── main.jsx        # Application entry point
```

## Customization

- Update `src/data/projectsData.js` to add your projects
- Update `src/data/skillsData.js` to add your skills
- Modify components in `src/components/` to customize the UI
